Marketplace and Catalog Operator
--------------------------------

For installations using the `Marketplace`_ operator installation method, you must install the `Marketplace and Catalog`_ operators. 
Although this method is optional, it offers the best possible integration with the Operator Lifecycle Manager. 


.. parsed-literal::

    git clone https://github.com/operator-framework/operator-marketplace.git 

    kubectl apply -f deploy/upstream

    kubectl rollout status -w deployment/marketplace-operator -n marketplace
    kubectl rollout status -w deployment/upstream-community-operators -n marketplace

.. _Marketplace and Catalog: https://github.com/operator-framework/operator-marketplace
